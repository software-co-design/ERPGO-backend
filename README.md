# ERPGO 系统开发使用说明

### 仓库获取

新建一个文件夹 ERPGO 作为你的本地仓库

初始化本地仓库

```bash
git init
```
与本远程仓库建立连接

```bash
git add origin https://github.com/Extreme-S/ERPGO-backend.git
```
抓取所有远程分支

```bash
git fetch
```
检出你所要用的分支，本项目 ERPGO 以 main 为主分支

```bash
git checkout -b main origin/main
```

到这一步就可以看到你的 ERPGO 文件夹下已经获取到代码了，打开IDEA开始编辑吧
